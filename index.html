Speedverse - Current Project

I've been working on Speedverse since 2019 as a dive-deep passion project. It's a multiplayer shooter game made using the Unreal Engine.



Working on Speedverse has given me exposure to many aspects of game development, such as the following..

Game Mechanics

Various first-person shooter gameplay mechanics using C++ and Unreal Engine's blueprint system with replication in mind.

Amazon GameLift/DynamoDB

The game server is deployed on Amazon GameLift to handle server auto-scaling and Flexmatch matchmaking rules. Player data such as character customization, wins, losses and more is stored in DynamoDB.

Modeling and Animation

I've created a number of 3D models with Blender and used Unreal's animation system to animate characters, weapons and various gameplay features.

Other

I've designed many materials using UE's material designer, worked on sound design and music using Ableton, created various icons and textures using Adobe Illustrator and Photoshop and created responsive UI elements using UE's UMG designer. I've also used Perforce for version control and implemented CI/CD for ease of development and testing.

Speedverse is a work in progress. I hope to show more soon!
